================================================================================
                              HOW TO USE
================================================================================

ADDING PRODUCTS WITH MULTIPLE IMAGES:
  1. Go to Admin Panel (/admin/login)
  2. Enter credentials: admin@tei.com / admin123
  3. Click "Products" menu
  4. Click "Add Product" button
  5. Fill form (name, category, description, specs)
  6. In "Product Images" section:
     - Upload multiple files at once
     - OR add images one-by-one via URL
     - See previews in grid
     - Hover to remove unwanted images
  7. Click "Save Product"
  8. Images immediately visible on frontend

VIEWING CAROUSEL ON PRODUCT PAGE:
  1. Go to any product detail page
  2. See large main image
  3. Click Previous/Next arrows to navigate
  4. Click thumbnails to jump to image
  5. See image counter (e.g., "2/5")
  6. On mobile: Swipe left/right (future)

EDITING EXISTING PRODUCT:
  1. Admin → Products
  2. Click edit (pencil) icon
  3. Add more images or remove existing ones
  4. Click "Save Product"
  5. Changes visible immediately

PRODUCT DATA STRUCTURE (NEW):
  type Product = {
    id: string
    name: string
    category: string
    subcategory: string
    description: string
    specifications: string[]
    images: string[]          // NEW: Array instead of single image
    featured: boolean
    createdAt: string
  }

CAROUSEL COMPONENT USAGE:
  <ProductCarousel 
    images={product.images} 
    productName={product.name} 
  />

IMAGE STORAGE:
  • Format: Base64 data URLs or external URLs
  • Location: Browser localStorage
  • Size: ~100-200KB per image average
  • Capacity: 50+ products × 5 images = 200 images total

TROUBLESHOOTING:
  Q: Images not uploading?
  A: Check file size (<2MB), format (JPG/PNG), try URL method

  Q: Images not showing on frontend?
  A: Hard refresh page (Ctrl+Shift+R), check localStorage enabled

  Q: Modal won't scroll?
  A: Scroll is inside content area (not modal body), try on mobile

  Q: Images lost after restart?
  A: Check localStorage enabled in browser settings
